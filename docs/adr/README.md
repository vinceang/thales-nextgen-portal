# Architecture Decision Records

Short, append-only records of architecturally significant decisions for the
Thales NextGen Portal app build. One file per decision, numbered.

Format: [Michael Nygard's ADR template](https://github.com/joelparkerhenderson/architecture-decision-record)
(Title · Status · Context · Decision · Consequences). Status moves
`Proposed → Accepted → (Superseded by NNNN)`; never edit a decided ADR's history —
supersede it with a new one.
