Thin single-weight line icon set (Lucide-equivalent). White, no fill — quiet and utilitarian.

```jsx
<Icon name="search" />
<Icon name="wifi" color="var(--color-highlight-blue)" />
```

- Names: menu, search, wifi, user, x, plane, chevron-right/left/down, sun, cloud, cloud-sun, cloud-rain.
- Default 22px, 1.75 stroke, currentColor. Imagery does the heavy lifting — keep icons sparse.
