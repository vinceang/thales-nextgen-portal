Bright-blue uppercase action label above a serif tile title (the only colored text on a tile).

```jsx
<Kicker>Destination</Kicker>
<h3 style={{ fontFamily: "var(--font-tile)" }}>Orlando's Best: Epcot</h3>
```

- Use the activity verb: CONNECT / WATCH / LISTEN / PLAY / READ / SHOP / DESTINATION / TRAVEL.
- `underline` gives the smaller linked variant.
