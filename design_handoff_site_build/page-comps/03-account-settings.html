Page Comp 03

Account / Settings

Fidelity Hi‑fi · Desktop 1200px · forms + tabs
Live render — Thales design system

Implementation spec

A settings surface that exercises every form control. Group fields in Cards; switch sections with Tabs.

1
Header + section Tabs

NavBar title="Account"; Tabs below with value/onChange (Profile · Connectivity · Billing). Active tab is bright‑blue with the underline.

2
Profile card — text fields

Inside a Card title="Profile": Input (label/placeholder/value/onChange), Select (options, placeholder), DatePicker (value:Date, onChange), TextArea (maxLength shows counter).

Layout: two‑column field grid (CSS grid, gap: var(--space-md)); full‑width for the textarea.

3
Preferences — toggles & choices

Toggle for instant on/off settings; RadioGroup (options, value, onChange) for one‑of‑many; Checkbox for opt‑ins.

Rule: Toggle = applies immediately; Checkbox = submitted with the form; Radio = mutually exclusive.

4
Actions row

Right‑aligned (justify-content:flex-end): secondary Button "Cancel" + primary "Save Changes". On save, fire a Toast tone="success".

•
Tokens & rules

All fields share the dark --surface-input fill, sharp corners, 1px --color-highlight-blue focus ring. Labels --font-sans. RTL‑safe (logical props) — mirror cleanly. Hit targets ≥44px.
