Three device tiers. Design width is the canvas you compose on; the collapse range is where the layout actually reflows (@media thresholds); type is driven by the fluid clamp() range 480→1200px.

Tier Design width Collapse range Layout Type
Desktop 1200px
--bp-desktop		 ≥ 1101px Full bento · 4-up posters · 3-up plans At max
Tablet 810px
--bp-tablet		 561 – 1100px 3-up posters · news 1-col · plans stack ≤810 Interpolating
Phone 390px
--bp-phone		 ≤ 560px Single column · 2-up posters · drawer nav At min (≤480px)

Prefer fluid --fs-* tokens over per-breakpoint font overrides — the scale handles size between tiers automatically. Reflow thresholds (1100 / 810 / 560) sit just below each design width so a tier holds its layout until clearly outgrown.
